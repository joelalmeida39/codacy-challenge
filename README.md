# codacy-challenge

This is a Codacy Challenge in React. 
The main goal is to fetch from the github API the list of commits for a specific repository. And with the option to search in this repository by specific fields.

# instructions
clone the project:
git clone https://github.com/joelalmeida39/codacy-challenge

make sure you have Node installed and updated.

go to the folder codacy-challenge/codacy-challenge and run:
npm install

and then:
npm start

It will be running in your http://localhost:3000
